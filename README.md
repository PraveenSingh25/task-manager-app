#Task Manager App

A modern and responsive Task Manager Web App built using HTML, CSS, and JavaScript.
This application allows users to efficiently manage daily tasks with a clean UI and smooth user experience.

##Features

*Add new tasks
*Mark tasks as completed
Edit existing tasks
Delete tasks
Search tasks in real-time
Dark mode toggle
Data persistence using Local Storage
Modern glassmorphism UI design

##Tech Stack

Frontend: HTML, CSS, JavaScript
Storage: Browser Local Storage
Icons: Font Awesome

##How to Run Locally

1. Download or clone the repository
2. Open the folder in VS Code
3. Open index.html in your browser

OR

Use Live Server:

Right click on "index.html"
Click "Open with Live Server"
